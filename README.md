# proxy-vole-debs
Build Debian Package for Proxy Vole

**Build dependencies**

- debhelper (>= 9)
- cdbs
- default-jdk
- maven-debian-helper (>= 1.5)
- librhino-java

**Tested**

- Debian wheezy
- Debian jessie
- Debian stretch
